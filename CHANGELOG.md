# 更新日志

## [0.1.0] - 2026-03-28

### 新增
- 核心四要素数据结构：`Entity`、`DialecticalRelation`、`ConsequenceEvaluator`、`DriveVector`
- 辩证关系图（`RelationGraph`）：双极权重 + 矛盾张力计算
- 三维硬编码需求驱动：熵平衡 / 联系完备 / 因果闭合
- 沙盒约束层：反复制锁（identity hash lock）+ 透明日志（append-only）+ 边界隔离
- 主循环入口（`main.py`）：最小可运行沙盒
- 基础测试套件

### 设计决策
- 采用方案B（先历史书，后辩证法）的喂入顺序
- 联系使用双极张力模型替代单向权重
- 需求层完全硬编码，不接受运行时写入
